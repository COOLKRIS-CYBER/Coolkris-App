# :video_game: CoolkrisCyber

เครื่องมือสำหรับสตรีมเมอร์ที่ช่วยให้คุณจัดการการสตรีมและมีปฏิสัมพันธ์กับผู้ชมได้อย่างมืออาชีพ ไม่ว่าคุณจะสตรีมบน TikTok หรือแพลตฟอร์มอื่นๆ

## :sparkles: ทำอะไรได้บ้าง

### :dart: จัดการการสตรีมอย่างง่าย
- **เชื่อมต่อกับ TikTok**: รับการแจ้งเตือนและข้อมูลการบริจาคแบบเรียลไทม์
- **วงล้อสุ่ม**: จัดกิจกรรมสุ่มเลือกผู้โชคดีจากผู้ชมในสตรีม
- **จัดการคะแนน**: บันทึกและแสดงคะแนนการแข่งขันในสตรีม
- **เซิร์ฟเวอร์ Minecraft**: สร้างและจัดการเซิร์ฟเวอร์สำหรับผู้ชม

### :closed_lock_with_key: ระบบสมาชิกที่ปลอดภัย
- **ล็อกอินง่ายๆ**: ใช้บัญชี Discord ในการเข้าสู่ระบบ
- **จัดการสิทธิ์**: เลือกฟีเจอร์ที่ต้องการใช้งานได้ตามความเหมาะสม
- **ต่ออายุการใช้งาน**: ขยายระยะเวลาการใช้งานได้ตามต้องการ

### :art: ใช้งานง่าย ดูสวยงาม
- **หน้าจอที่ใช้ง่าย**: ออกแบบมาให้ใช้งานได้ง่ายสำหรับทุกคน
- **โหมดสว่าง/มืด**: เลือกธีมที่ถูกใจได้ตามต้องการ
- **อัปเดตแบบเรียลไทม์**: ข้อมูลทุกอย่างอัปเดตทันทีทันใด

## :rocket: การเริ่มต้นใช้งาน

### ดาวน์โหลดและติดตั้ง
1. ดาวน์โหลดไฟล์ติดตั้งจาก [Releases](https://github.com/COOLKRIS-CYBER/Coolkris-App/releases)
2. รันไฟล์ติดตั้งและทำตามขั้นตอน
3. เปิดโปรแกรมและเริ่มใช้งานได้เลย!

### การเริ่มต้นครั้งแรก
1. เปิดโปรแกรม CoolkrisCyber
2. ล็อกอินด้วยบัญชี Discord ของคุณ
3. เลือกฟีเจอร์ที่ต้องการใช้งาน
4. เริ่มสตรีมและสนุกกับการทำงานได้เลย!

## :bulb: คำแนะนำการใช้งาน

### สำหรับสตรีมเมอร์มือใหม่
- เริ่มต้นจากฟีเจอร์พื้นฐานก่อน เช่น การรับการแจ้งเตือนจาก TikTok
- ลองใช้วงล้อสุ่มเพื่อเพิ่มความสนุกในสตรีม
- ใช้ระบบคะแนนเพื่อสร้างการแข่งขันในชุมชน

### สำหรับสตรีมเมอร์ที่มีประสบการณ์
- ใช้ฟีเจอร์การจัดการสมาชิกเพื่อควบคุมการเข้าถึง
- เชื่อมต่อหลายแพลตฟอร์มพร้อมกันเพื่อขยายฐานผู้ชม
- ใช้เซิร์ฟเวอร์ Minecraft เพื่อสร้างชุมชนที่แข็งแกร่ง

## :dart: ทำไมต้อง CoolkrisCyber

- **ใช้งานง่าย**: ไม่ต้องมีความรู้ด้านเทคนิคก็ใช้ได้
- **รวดเร็ว**: ทำงานได้รวดเร็วแม้บนคอมพิวเตอร์ทั่วไป
- **ปลอดภัย**: ข้อมูลของคุณปลอดภัยด้วยระบบรักษาความปลอดภัยขั้นสูง
- **ฟรี**: ใช้งานได้ฟรีไม่มีค่าใช้จ่ายแอบแฝง
- **อัปเดตตลอด**: มีการอัปเดตและปรับปรุงฟีเจอร์อย่างสม่ำเสมอ
